# Lane Marking Segmentation using Image Processing (MATLAB)

This project implements a classical computer vision pipeline to detect road lane markings from driving scene images.

## Methods
- HSV color segmentation
- White and yellow lane detection
- Canny edge detection
- Morphological filtering
- Region of interest masking

## Output
Binary segmentation masks and visualization overlays highlighting detected lane markings.

## Technologies
- MATLAB
- Image Processing Toolbox
